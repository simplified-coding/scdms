Καλώς ήρθατε στο Simplified Coding Data Management System!

Η Υπηρεσία Διαχείρισης Δεδομένων Simplified Coding ή SCDMS είναι η κεντρική υπηρεσία της Simplified Coding που διαχειρίζεται όλες τις διεργασίες σχετικά με τα δεδομένα σας. Οι υπηρεσίες που προσφέρονται κυμαίνονται από υπηρεσίες μόνο για διαχειριστές, όπως η διαχείριση πιστοποιητικών, έως υπηρεσίες για κανονική χρήση, όπως η οριστική διαγραφή δεδομένων.
